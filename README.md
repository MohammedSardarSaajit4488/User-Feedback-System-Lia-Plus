# 🧠 User-Feedback-System-lia-plus - Full Stack App
✨ Full-stack feedback system built using React, Node.js, Express, and MongoDB for Lia Plus SDE1 technical assignment.

A professional full-stack feedback system built using **React**, **Node.js**, **Express**, and **MongoDB**. Users can submit feedback, view previous submissions, and categorize them into suggestions, bugs, or feature requests — all in a modern, responsive UI.

## 🚀 Features

- 📥 Submit feedback with name, email, category, and message
- 📂 Auto-group feedbacks by category (Suggestion, Bug Report, Feature Request)
- 🧾 Clean card-style feedback dashboard
- ❌ Delete feedback with one click
- 🔄 Auto-refresh dashboard after submission
- 🧼 Responsive UI using HTML + CSS (no frameworks)
- 💡 Powered by MongoDB Atlas cloud database

## 🛠️ Getting Started – Run the App Locally

Follow these steps to run the full-stack application in your local machine using VS Code.

# Install Dependencies:

- cd frontend
- npm install
- cd ..

# START APPLICATION:
- cd backend
- node server.js

- cd frontend (NEW TERMINAL)
- npm start
